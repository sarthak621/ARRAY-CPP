<h2><a href="https://www.geeksforgeeks.org/problems/shortest-path-in-weighted-undirected-graph/1">Shortest Path in Weighted undirected graph</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p data-start="397" data-end="581"><span style="font-size: 14pt;">You are given a weighted undirected graph with <strong><code data-start="444" data-end="447">n</code></strong> vertices numbered from <strong><code data-start="471" data-end="474">1</code></strong> to <strong><code data-start="478" data-end="481">n</code> </strong>and <strong><code data-start="486" data-end="489">m</code></strong> edges along with their weights. Find the <strong>shortest path</strong> between vertex <strong><code data-start="560" data-end="563">1</code></strong> and vertex <strong><code data-start="575" data-end="578">n</code></strong>.&nbsp;<span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">Each edge is given as&nbsp;</span><strong style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"><code data-start="865" data-end="876">{a, b, w}</code></strong><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">, denoting an edge between vertices&nbsp;</span><code data-start="912" data-end="915">a</code><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">&nbsp;and&nbsp;</span><code data-start="920" data-end="923">b</code><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">&nbsp;with weight&nbsp;</span><code data-start="936" data-end="939">w</code><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">.</span></span></p>
<p data-start="583" data-end="841"><span style="font-size: 14pt;">If a path exists, return a list of integers where the <strong data-start="637" data-end="654">first element</strong> is the <strong data-start="662" data-end="678">total weight</strong> of the shortest path, and the <strong data-start="709" data-end="731">remaining elements</strong> are the <strong data-start="740" data-end="749">nodes</strong> along that path (from <code data-start="772" data-end="775">1</code> to <code data-start="779" data-end="782">n</code>). If no path exists, return a list containing only <strong>{<code data-start="836" data-end="840">-1}</code></strong>.</span></p>
<p><span style="font-size: 14pt;"><strong>Note:&nbsp;</strong><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">The driver code will internally verify your returned list.</span></span></p>
<ul>
<li><span style="font-size: 14pt;"> If both the path and its total weight are valid, only the total weight will be displayed as output.</span></li>
<li><span style="font-size: 14pt;">If your list contains only&nbsp;<code data-start="517" data-end="521">-1</code>, the output will be&nbsp;<code data-start="542" data-end="546">-1</code>.</span></li>
<li><span style="font-size: 14pt;">If the returned list is invalid, the output will be <code data-start="482" data-end="486">-2</code>.</span></li>
</ul>
<p><span style="font-size: 14pt;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>n = 5, m= 6, edges = [[1, 2, 2], [2, 5, 5], [2, 3, 4], [1, 4, 1], [4, 3, 3], [3, 5, 1]]
<strong>Output: </strong>5
<strong>Explanation: </strong>Shortest path from 1 to n is by the path 1 4 3 5 whose weight is 5. <br></span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>n = 2, m= 1, edges = [[1, 2, 2]]
<strong>Output: </strong>2
<strong>Explanation: </strong>Shortest path from 1 to 2 is by the path 1 2 whose weight is 2. </span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>n = 2, m= 0, edges = [ ]
